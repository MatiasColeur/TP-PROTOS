# POROTOS

Este proyecto implementa un **servidor SOCKS5 no bloqueante**.

## Requisitos
Antes de ejecutar asegurarse de tener instalado:
- **gcc**
- **Make**
- **openssl**


## Compilación
```bash
$> make clean all
```

## Ejecución
Ejecución del servidor socks5:

```bash
$> ./bin/socks5
```

Ejecución del cliente:

```bash
$> ./bin/client
```

Ejecución de la API: 

```bash
$> ./bin/api
```
